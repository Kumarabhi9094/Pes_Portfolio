## 🚀 Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)
- No server or build tools required

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/yourusername/portfolio.git
   cd portfolio
   ```

2. **Open locally**
   ```bash
   # Simply open the HTML file in your browser
   # Or use a local server
   python -m http.server 8000
   # Then visit http://localhost:8000
   ```

3. **For better development**
   - Install VS Code Live Server extension
   - Right-click `index.html` → "Open with Live Server"

## 📝 Customization

### Update Your Information
1. Open `index.html` and update:
   - Name and professional summary
   - Skills and technologies
   - Projects and links
   - Contact information
   - Social media links (LinkedIn, GitHub)

### Add Profile Photo
- Replace `photo-placeholder` section with an `<img>` tag pointing to your photo

### Update Certificates
- Add new certificate files to `/certificate` folder
- Update the certificate cards in the certificates section

### Modify Styling
- Edit `styles.css` for colors, fonts, and layout
- Use CSS variables defined in `:root` for easy theme customization

### Add Analytics (Optional)
- Integrate Google Analytics or similar tracking
- Add tracking code to `<head>` section of `index.html`

## 🌐 Deployment

### GitHub Pages (Free & Recommended)
1. Push your code to GitHub
2. Go to repository **Settings** → **Pages**
3. Select `main` branch as source
4. Your site will be live at `https://yourusername.github.io/portfolio`

### Other Options
- **Netlify** - Connect GitHub repo, auto-deploys on push
- **Vercel** - Similar to Netlify with great performance
- **Traditional Hosting** - Upload files via FTP to any web host
